### 使用脚本的好处
1. 脚本能迅速筛选出所有符合条件的全场最佳座位，
2. 能几乎同时对所有“全场最佳”发起锁定请求，无论哪一个请求成功，都是达到目的。

### 备注
对于"XXOOOXOXOOXO"(X表示不可预定，O表示可以预定)，在需要2个座位的前提下，有三种选择。

### 关于选座(cosy zone)
会优先选择靠中间的排，再在每排选择靠中间连续的座位。

### 对“全场最佳”的解释
中心及中心靠后。

### 注意事项
1. 淘票票目前普遍是单次最多选4个座位，脚本会优先锁定4个连在一起的座位（默认配置），大于4个的需求无法保证所有座位是连续的。
